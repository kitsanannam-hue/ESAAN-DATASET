
# Data Directory

This directory contains raw and processed data files.

## Structure

- `raw/` - Original, unmodified data files
  - `dissertations/` - PhD dissertations and research papers
  
- `processed/` - Cleaned and processed data files
  - Generated datasets
  - Intermediate processing results

## Note

The `output/` directory at the project root contains the final exported datasets.
This `data/` directory is for source materials and intermediate processing.
